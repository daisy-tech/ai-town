# 记忆数据浏览器使用说明

## 📁 文件位置

- **后端API**: `convex/memoryViewer.ts`
- **前端页面**: `tools/memory-browser.html`

## 🔌 连接真实数据

目前浏览器显示的是模拟数据。要连接真实数据，需要集成Convex客户端：

### 方法1：使用Convex React（推荐）

将 `memory-browser.html` 转换为React组件：

```tsx
import { useQuery } from 'convex/react';
import { api } from '../convex/_generated/api';

export function MemoryBrowser() {
  const stats = useQuery(api.memoryViewer.getMemoryStats);
  const memories = useQuery(api.memoryViewer.getAllMemories, {
    playerName: selectedPlayer,
    memoryType: selectedType,
    limit: PAGE_SIZE,
    offset: currentPage * PAGE_SIZE,
  });

  // ... 使用真实数据渲染
}
```

### 方法2：直接使用HTTP API

在 `memory-browser.html` 中修改API调用：

```javascript
// 安装Convex客户端
// npm install convex

import { ConvexHttpClient } from "convex/browser";

const client = new ConvexHttpClient("YOUR_CONVEX_URL");

async function loadMemories() {
  const result = await client.query(api.memoryViewer.getAllMemories, {
    playerName: playerFilter,
    memoryType: typeFilter,
    limit: PAGE_SIZE,
    offset: currentPage * PAGE_SIZE,
  });

  displayMemories(result);

```

### 方法3：通过npx convex命令

在终端中直接查询：

```bash
# 查看所有记忆
npx convex run memoryViewer:getAllMemories '{}'

# 查看统计信息
npx convex run memoryViewer:getMemoryStats '{}'

# 查看特定角色的记忆
npx convex run memoryViewer:getAllMemories '{"playerName":"迪迦奥特曼"}'

# 查看对话类型的记忆
npx convex run memoryViewer:getAllMemories '{"memoryType":"conversation"}'
```

## 📊 API参考

### getAllMemories

查询记忆列表

**参数**:
```typescript
{
  playerName?: string,     // 角色名称（可选）
  memoryType?: 'conversation' | 'reflection' | 'relationship',  // 记忆类型（可选）
  limit?: number,          // 每页数量（默认50）
  offset?: number,         // 偏移量（默认0）
}
```

**返回**:
```typescript
{
  memories: Memory[],      // 记忆列表
  total: number,           // 总数量
  offset: number,          // 当前偏移
  limit: number,           // 每页数量
}
```

### getMemoryStats

获取统计信息

**返回**:
```typescript
{
  total: number,                              // 总记忆数
  typeStats: {                                // 类型统计
    conversation: number,
    reflection: number,
    relationship: number,
  },
  playerStats: Array<{                        // 角色统计
    playerName: string,
    totalMemories: number,
    conversations: number,
    reflections: number,
    avgImportance: string,
  }>,
  avgImportance: string,                      // 平均重要性
  recentMemories: Memory[],                   // 最近的5条记忆
}
```

### getMemoryDetail

获取记忆详情

**参数**:
```typescript
{
  memoryId: Id<'memories'>   // 记忆ID
}
```

## 💡 扩展建议

### 添加搜索功能

在控制面板中添加文本搜索：

```javascript
async function searchMemories(keyword) {
  const result = await client.query(api.memoryViewer.getAllMemories, {});
  const filtered = result.memories.filter(m =>
    m.description.includes(keyword)
  );
  displayMemories({ memories: filtered, total: filtered.length });
}
```

### 添加导出功能

导出为CSV或JSON：

```javascript
function exportToCSV(memories) {
  const csv = memories.map(m =>
    `"${m.playerName}","${m.type}","${m.description}","${m.importance}","${m.createdAt}"`
  ).join('\n');

  const blob = new Blob([csv], { type: 'text/csv' });
  const url = URL.createObjectURL(blob);
  const a = document.createElement('a');
  a.href = url;
  a.download = 'memories.csv';
  a.click();
}
```

## 🐛 故障排除

### 问题1: 显示"加载记忆数据中..."一直不消失

**原因**: API调用失败或Convex未连接

**解决**:
1. 确保Convex dev服务器正在运行 (`npm run dev`)
2. 检查浏览器控制台错误信息
3. 确认Convex URL配置正确

### 问题2: 记忆列表为空

**原因**: 游戏中还未生成记忆

**解决**:
1. 让游戏运行一段时间
2. 确保角色之间有对话
3. 使用 `npx convex run memoryViewer:getMemoryStats '{}'` 检查数据库

### 问题3: 样式显示异常

**原因**: 浏览器兼容性问题

**解决**:
1. 使用现代浏览器（Chrome、Firefox、Edge）
2. 确保启用了CSS Grid和Flexbox支持
3. 检查backdrop-filter支持（部分浏览器需要前缀）
